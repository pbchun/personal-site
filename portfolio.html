Projects

Bark Park

Bark Park is a mobile app that allows users to see which dogs are currently at their favorite dog park. Users can see a list of dog parks and sign-in and out of a dog park. Users can also create a profile for each of their dogs and add their favorite dog park if it's not already on the list. This mobile app was created using React Native for the front-end, and Express, Knex.js, PostgreSQL, and Node.js for the back-end. This project was part of my capstone project while at Galvanize and I am currently working on it so that it will be a fully functioning app.

LiveWire

Livewire is a web app that allows venue and event promoters to list available dates for artists to request to book. The user can also see a list of upcoming events by event venue. This project was created as part of a group project while at Galvanize using Vue.js for the front-end, and Express, Knex.js, PostgreSQL, and Node.js for the back-end.

Foreign Currency Exhange Rate App

This web app allows the user to see the most recent foreign currency exchange rate compared to $1 (USD). The user can then select any date since 1999 to compare how the rate has changed. The exchange rates are provided by the European Central Bank from the exchangeratesapi.io API. This app was created using HTML, CSS, and JavaScript.

What's Your Dollar Worth GitHub link

Trip Budget Tracker App

This web app allows users to keep track of their budget for upcoming trips or vacations. The user can add a new trip and edit their trip budget as it changes. This app was created using React for the front-end and Express, Knex.js, PostgreSQL, and Node.js for the back-end.
